#  Modern Tic Tac Toe (Python Tkinter)

A stylish and interactive Tic Tac Toe game built using **Python**, **Tkinter**, and **NumPy**. This version features a modern dark UI, score tracking, and a simple AI opponent.

---

##  Features

*  Modern dark-themed GUI
*  Computer opponent with basic AI logic
*  Smart move strategy:

  * Wins if possible
  * Blocks player winning moves
  * Otherwise plays randomly
*  Live score tracking (Player vs Computer)
*  Restart game functionality
*  Detects:

  * Win
  * Draw
*  Smooth gameplay with delayed computer moves

---

##  Technologies Used

* **Python 3**
* **Tkinter** (GUI)
* **NumPy** (board management)
* **Random module** (AI move selection)

##  How to Play

* You are **Player (X)**
* The computer is **O**
* Click on any empty cell to make your move
* The computer will respond automatically
* First to get 3 in a row (horizontal, vertical, or diagonal) wins

---

##  Game Logic

The computer follows a simple strategy:

1. Check if it can **win immediately**
2. Block the player's **winning move**
3. Otherwise, pick a **random available cell**
